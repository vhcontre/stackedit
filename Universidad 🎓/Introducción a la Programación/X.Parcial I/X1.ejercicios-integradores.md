## üêç **C√≥digo Python - Ejercicio Integrador 1**

```python
# Ejercicio 1: Estad√≠sticas de N√∫meros

positivos = 0
negativos = 0
suma_positivos = 0
suma_negativos = 0
cantidad_negativos = 0

for i in range(10):
    numero = int(input(f"Ingrese el n√∫mero {i + 1} (no puede ser 0): "))
    while numero == 0:
        print("El n√∫mero no puede ser cero.")
        numero = int(input(f"Ingrese nuevamente el n√∫mero {i + 1}: "))

    if numero > 0:
        positivos += 1
        suma_positivos += numero
    else:
        negativos += 1
        suma_negativos += numero
        cantidad_negativos += 1

print(f"\nTotal de positivos: {positivos}")
print(f"Total de negativos: {negativos}")
print(f"Suma de positivos: {suma_positivos}")
if cantidad_negativos > 0:
    promedio_negativos = suma_negativos / cantidad_negativos
    print(f"Promedio de negativos: {promedio_negativos:.2f}")
else:
    print("No se ingresaron n√∫meros negativos.")

```

### ‚úèÔ∏è Explicaci√≥n:

-   Se usa un `for` para garantizar que se ingresen exactamente 10 n√∫meros.
    
-   Se valida que los valores ingresados no sean cero.
    
-   Se usan contadores y acumuladores para registrar positivos y negativos.
    
-   Al final, se muestra un resumen de todos los valores calculados.
